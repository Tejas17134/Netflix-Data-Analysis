# Netflix Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset to understand content distribution, ratings, and global production trends.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Insights
- Movies dominate the Netflix platform compared to TV Shows.
- Most content is rated TV-MA.
- Content additions increased significantly after 2016.
- The United States produces the most Netflix content.

## Dataset
Netflix Movies and TV Shows Dataset.
